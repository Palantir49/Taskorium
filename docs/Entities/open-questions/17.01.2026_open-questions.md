### Вопрос #2: Как пересекаются доступы между project и workspace?
- **Варианты**:
    - **Добавить роль участник** — отдельная роль, которая будет использоваться для worksapace, в случае если у пользователя есть доступ только к project 
    - **Давать пользователю доступ конкретно к сущности** —  Если у пользователя доступ только к проекту, то у рабочей области нет связи.
- **Влияние:**
	- **Добавить роль участник** - будет полезно в аналитике, если будет аналитика по проекту
